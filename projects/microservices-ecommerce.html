🛒 Microservices E-Commerce Prototype

Overview

A minimal e-commerce prototype with polyglot microservices (Rust, Node.js, Python, Go), SSR frontend, and robust CI/CD. Features product catalog, user management, order processing, and recommendation service.

Core Features

  • 🛍️ Product catalog (SSR)
  • 🛒 Cart & checkout UI
  • 👤 User authentication
  • 📦 Order processing
  • 🤖 AI-powered recommendations
  • 🧑‍💻 Admin panel (optional)
  • 🔔 Notifications & metrics
  • 🔒 Security & rate limiting

Tech Stack

  • Next.js
  • React
  • TypeScript
  • Tailwind CSS
  • Rust/Node.js/Python/Go
  • PostgreSQL
  • MongoDB
  • Redis
⭐ Polyglot, scalable, and ready for interview discussion.
