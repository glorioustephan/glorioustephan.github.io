🍳 AI-Powered Recipe Collector

Status: Design Stage

Overview

A full-stack web app designed to collect, extract, and organize online recipes using AI-powered web scraping and content parsing. Users submit recipe URLs, which are scraped and processed by an AI API to normalize and store the recipes in a structured format. Features include search, filtering, categorization, and dynamic recipe views—all built for modern systems design practice.

Core Features

  • 🌐 URL-based recipe ingestion with web scraping
  • 🧠 AI-powered recipe extraction and normalization
  • 🗃️ Recipe database with search, filtering, and faceting
  • 📊 Categorization and tagging system
  • 🔎 Full-text search and filtering UI
  • ⚡ End-to-end full stack architecture for systems design practice
  • 🚀 Deployment on modern serverless platforms (Vercel, Neon, Upstash)
  • 🛠️ Modular design to simulate microservices and scalable patterns

Tech Stack

  • Next.js 14+
  • React
  • TypeScript
  • Tailwind CSS
  • Node.js (API)
  • PostgreSQL (Neon or Supabase)
  • Redis (Upstash)
  • OpenAI / Claude API
⭐ Modern AI-powered content extraction and full-stack design for systems architecture practice.
