📝 SEO-Optimized Blog Platform

Overview

A multi-author blog/CMS platform with Markdown/rich editor, SSR/ISR for SEO, and AI-powered tag and summary suggestions. Includes admin/editor UI, public blog, analytics, and newsletter integration.

Core Features

  • 📰 Public blog pages (SSR/ISR)
  • 🧑‍💻 Author dashboard
  • 🏷️ AI-powered tag suggestions
  • 🔍 Search, RSS, sitemap, analytics
  • ✉️ Newsletter signup (optional)
  • 🔒 Role-based access, JWT/session auth

Tech Stack

  • Next.js
  • React
  • TypeScript
  • Tailwind CSS
  • Python/Node.js
  • PostgreSQL
  • Redis
  • MongoDB
